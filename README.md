# IoT-Based Air Quality & Pollution Monitoring Dashboard

This is a professional IoT project that measures air quality parameters in real time, transmits them to a cloud dashboard, and alerts users when pollution exceeds safe limits.

## 📁 Structure
* `arduino_code/`: Firmware for ESP32/Arduino.
* `python_simulation/`: Python script to simulate environmental data.
* `data/`: Automated CSV data logs.
